# Javier Cancela Blog

Personal Jekyll site for blog posts and static pages.

## Development

- Install dependencies with `bundle install`
- Start the local site with `bundle exec jekyll serve --livereload`
- Build the site with `bundle exec jekyll build`
- Check the site configuration with `bundle exec jekyll doctor`

## Content Notes

- Keep post front matter minimal and consistent
- Do not add legacy `subtitle` or header `image` fields to posts
- Store post assets under `/assets/` and reference them with absolute paths
